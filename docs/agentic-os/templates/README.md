# Templates Executáveis

## 1) PRD (O QUÊ / PORQUÊ)
Campos mínimos:
- objetivo
- contexto
- problema
- hipótese
- escopo / fora de escopo
- critérios de aceitação
- métricas e baseline
- `[NEEDS CLARIFICATION]`

## 2) RFC
Campos mínimos:
- arquitetura alvo
- fluxos de dados
- contratos/APIs
- observabilidade
- segurança
- deploy/rollback
- impacto sistêmico

## 3) ADR Temporal
Campos mínimos:
- contexto
- decisão
- trade-offs
- alternativas descartadas
- `valid_from`
- `valid_until`
- `review_trigger`

## 4) RCA com CIG
Campos mínimos:
- timeline
- cadeia causal
- causa raiz
- impacto
- correção imediata
- prevenção
